🍔 Food Delivery Web Application

A scalable and modern food delivery web application built using React 19 and Vite, designed with a focus on performance, maintainability, and user experience.

🚀 Live Demo

👉 link here https://johnson693.github.io/food-delivery-app/

📌 Overview

This project is a frontend-focused full-stack ready application that simulates a real-world food ordering system. It enables users to browse food items, manage cart operations, and navigate seamlessly through the application.

The application follows a component-based architecture and implements efficient state management using React Context API.


🧰 Tech Stack
Frontend
React 19 (with React Compiler)
Vite 8
React Router v7
HTML5, CSS3, JavaScript (ES6+)
Tools & Practices
Context API (State Management)
ESLint (Code Quality)
Modular Component Architecture
Responsive Web Design


✨ Key Features
🛒 Dynamic Cart System
Add/remove items with real-time updates
Automatic total price calculation
🔐 Protected Routing
Secure navigation for authenticated users
⚡ Optimized Performance
Memoization using useMemo and useCallback
📱 Responsive UI
Fully optimized for mobile, tablet, and desktop
♻️ Reusable Components
Navbar, Footer, Product Cards, Cart System
🔄 Client-Side Routing
Seamless navigation using React Router



🏗️ Architecture

Centralized state management using Context API
